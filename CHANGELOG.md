# Changelog

All notable changes to Enix Addons for Elementor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0] - 2024-05-07

### Added
- **Advanced Accordion Widget** - Interactive accordion with smooth animations and customizable styling
- **Timeline Steps Widget** - Process visualization widget with vertical/horizontal layouts
- **Faded Heading Widget** - Typography effects widget with gradient and fade animations
- **Inline Link Preview Widget** - URL preview widget with rich metadata extraction
- **Weather Widget** - Real-time weather data display with location support
- **Professional Plugin Structure** - Proper WordPress/Elementor integration
- **Comprehensive Asset Management** - Optimized CSS and JavaScript loading
- **WordPress Coding Standards** - Full compliance with WordPress best practices
- **Elementor Compatibility** - Tested with Elementor 3.21.0+
- **Multilingual Support** - Ready for translation with proper text domain setup

### Technical Features
- FontAwesome 6 integration for icons
- Responsive design for all widgets
- Accessibility compliance (WCAG 2.1)
- Performance optimized asset loading
- Proper security validation and sanitization
- Cross-browser compatibility testing
